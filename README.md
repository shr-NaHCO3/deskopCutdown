# deskopCutdown
页面的代码没有任何可读性。可作为反面教材。

# 使用方式

> 在编译之前，请确保你的电脑上拥有`node.js` `npm`

1. 安装依赖项

   ```
   npm i -g
   ```
   
      > 也可以使用 ```cnpm i -g```或```yarn```

<br/>

2. 试运行

    ```
    electron .
    ```
    
      >如果你修改了代码，在打包之前确保代码达到了你满意的效果


<br/>

3. 打包

    ```
    npm run dist1
    ```
    
      > 条件允许的话，使用`yarn`或`cnpm`代替`npm`.
